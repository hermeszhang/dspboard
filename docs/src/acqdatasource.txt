The AcqDataSource is the Filterlink component that the acqboard
interfaces with. There are three commands we can send: 


0x40: query: 
0x41: set

For set:  
DATA[0] : parameter #
DATA[1] : chan mask
DATA[2] : value

AcqDataSource Update (bcast, CMD = 0x60)
For linkstatus:
Data[0] = 0
data[1] = 1 if up, 0 if down

data[0] : type
data[1] : channel
data[2] : gain
data[3] : HPF setting
data[4] : cont setting...


type :
0 : LINK STATUS changed
1 : Channel state
2 : mode state
